# Holiday Card 2025

An interactive holiday card featuring Anakaren, Aaron, and their cats Sente & Gote.

## Features

- Scrollable sections with photos and stories from the year
- Cat animations that trigger as you scroll
- Background music
- Visual card builder for prototyping different layouts

## Development

```bash
npm install
npm run dev
```

## Builder

Visit `/builder.html` to use the visual card builder:

- Edit intro text (year, title, from line)
- Add/remove sections with different layouts
- Upload images and adjust tilt/size
- Choose cat animations per section
- Select background music (default, silent, or custom)
- Export config as JSON or generate share links

## Tech Stack

- Vite
- Vanilla JS (no framework)
- CSS custom properties for theming

## Deployment

Configured for Vercel:

```bash
vercel
```
